🧠 Neuron Chat

Chat together, Dream together.

Neuron is a real-time chat application where users can send messages, share media, and interact with others.

✨ Features
💬 Real-time messaging
👍 Like / 👎 Dislike messages
↩️ Reply to messages
🧑‍💻 Admin system (Ray)
🔒 Login system
🚀 How to Run
Install Node.js
Run:
npm install
node server.js
Open:
http://localhost:3000
👑 Admin

User Ray has special permissions:

Delete messages
Ban/unban users
Make announcements
🎨 Theme
Black & white UI
Minimalist design
⚠️ Note

This is a starter project and can be expanded with:

Database
File uploads
Better security
